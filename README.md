# YTDL-TERMUX
Using this tool you can easily Download YouTube, Facebook, Instagram videos and audio using Termux.

## Usage:
### First Do Setup:
```bash
pkg install wget

apt install wget -y

wget https://github.com/ImBhashitha/YTDL/raw/main/setup.sh

chmod +x setup.sh

./setup.sh
```
